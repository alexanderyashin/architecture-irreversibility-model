# Paper Package

This folder contains the manuscript associated with the
**Architecture Irreversibility Model**.

The manuscript is intended for:
- Zenodo publication
- archival citation
- reference in research and professional discussions

---

## Scope

The paper:
- introduces a quantitative threshold-based model of irreversibility,
- explains cascade amplification via dependencies,
- links architectural states to expected economic loss,
- and demonstrates the approach on synthetic scenarios.

---

## Reproducibility

All figures and results referenced in the manuscript
are reproducible using the code in this repository.

No proprietary data or calibration logic is used.
